Apna-Hisab
Apna-Hisab is a comprehensive, fully responsive personal finance management web application. It is designed to help users track their daily transactions, manage budgets, and visualize their financial health with a seamless experience across mobile, tablet, and desktop devices.

Key Features
Responsive Web Design: A fluid user interface that adapts dynamically to any screen resolution and system environment.

Real-time Calculations: Instant updates on total balance, income, and expenditure.

Transaction History: Organized logs of all past financial activities.

Clean & Intuitive UI: Built with a focus on user experience and minimalist aesthetics.

Tech Stack
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)

Backend: Java

Build Tool: Maven
